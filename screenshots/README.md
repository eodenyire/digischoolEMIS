
## Regenerating Screenshots

Screenshots are generated by the `scripts/capture_screenshots.py` script.
The API must be running first (locally, via Docker, or via Kubernetes).

```bash
# Option 1 — local
make local-api          # in one terminal
make screenshots        # in another terminal

# Option 2 — Docker
make docker-up
make screenshots

# Option 3 — Kubernetes (minikube)
make k8s-deploy
make k8s-port-forward   # in one terminal
make screenshots        # in another terminal
```

## End-to-End Test Results

All Wave 1 API tests pass (functional + integration/regression across all 20 modules):

```
tests/test_api.py         — all functional tests passed
tests/test_integration.py — all integration tests passed
```

Run the tests yourself:

```bash
cd services/wave1-fastapi
pip install -r requirements.txt
python -m pytest tests/ -v
```

## Running the System

### Locally

```bash
make local-install
make local-api        # API on http://localhost:8000
make local-test       # run all tests
```

### Docker

```bash
make docker-up        # starts postgres + wave1-api
make docker-ps        # check container status
make docker-logs      # tail logs
make docker-test      # run tests inside container
```

### Kubernetes (minikube)

```bash
minikube start
make k8s-build        # build image inside minikube
make k8s-deploy       # apply manifests
make k8s-status       # check pod status
make k8s-port-forward # forward API to localhost:8000
```
